CHESS PUZZLE API

Chess puzzle API! This API utilizes chess puzzle data from the lichess database. We will continue to update, so please give us a lot of interest.

Notice

  • Limit values per page maximum is 100
  • Using JWT to authentication
  • There is a 5 second delay between requests.
    • When on delay, the response is (401, unauthorized)
