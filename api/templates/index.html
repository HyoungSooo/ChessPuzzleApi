Chess Puzzle API



Default Url : "https://chess.run.goorm.io"


  • GET /puzzle?limit=<int>& offset=<int> ⇒ Provides puzzle data as much as limit.
{
  "items": [
    {
      "puzzleid": "string",
      "rating": "string",
      "fen": "string",
      "tag": "string",
      "gameurl": "string"
    }
  ],
  "count": 0
}
  • GET /puzzle/move?puzzleid=<str> ⇒ Provides problem information for specific puzzles
}
	"puzzleid": "string",
  "tag": "string",
  "fen": "string",
  "move": [
    "string"
  ]
}
  • GET /theme ⇒ Provides themes for all puzzles
[
  {
    "theme": "string"
  }
]
  • GET /theme/puzzle?theme=<str> ⇒ Provides one random puzzle among puzzles with a specific theme
{
  "puzzleid": "string",
  "tag": "string",
  "fen": "string",
  "move": [
    "string"
  ]
}
